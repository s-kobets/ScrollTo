# Описание
	
  Представленный код, дает возможность использовать скролл,
  так чтобы происходил переход от блока к блоку, 
  не убирая полосу прокрутки страницы и учитывая положение 
  ползунка.

	Пример использования в index.html

# Install

	npm i

# Start

	npm run start

# Build

	npm run build
